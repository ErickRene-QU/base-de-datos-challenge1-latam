## 📝 Descripción del Proyecto
Este proyecto forma parte del **Challenge ONE** (Oracle Next Education y Alura Latam). El objetivo principal es resolver un caso de estudio real de Inteligencia de Negocios (BI) y Análisis de Datos para "Alura Store". 

El propietario, el señor Juan, posee cuatro sucursales diferentes y necesita tomar la decisión estratégica de **vender una de ellas** para invertir en un nuevo negocio. A través de este análisis de datos, identificamos objetivamente cuál es la tienda con el menor desempeño.

## 🎯 Aspectos Analizados
El análisis de los conjuntos de datos de las cuatro tiendas se centró en 5 métricas clave de negocio:
1. **Facturación total:** Cálculo de los ingresos generados por cada sucursal.
2. **Ventas por categoría:** Identificación de los nichos de mercado más populares.
3. **Valoración media:** Análisis de la satisfacción del cliente (1 a 5 estrellas).
4. **Productos más y menos vendidos:** Rendimiento individual del catálogo de productos.
5. **Costo promedio de envío:** Impacto de la logística en el rendimiento de la tienda.

🌟 **Reto Extra:** Se implementó un análisis geográfico utilizando coordenadas (latitud y longitud) para mapear la distribución de las ventas de las tiendas.

* **Visualización:** Matplotlib (Gráficos de barras, pastel y dispersión geográfica).
